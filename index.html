Hodgkin-Huxley Simulation

This simulation uses the Hodgkin-Huxley model to explore nerve communication. Adjust parameters to see their impact on action potentials. It includes preset scenarios for ease of use.

The default Dormand-Prince solver balances accuracy with larger time steps. Alternatives like Runge-Kutta and Euler are available for faster, though less precise, results. Modify the time step and duration to alter simulation speed and length. Input currents are set by listing time, duration, and strength.

Change parameters or select a scenario from the dropdown to update the plot automatically. Click "Update Plot" after adjustments to view changes.
